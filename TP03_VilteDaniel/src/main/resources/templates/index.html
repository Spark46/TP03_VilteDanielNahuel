Sección 1

Java

Java es un lenguaje de programación de alto nivel y orientado a objetos, desarrollado por Sun Microsystems (ahora propiedad de Oracle). Es conocido por su portabilidad, lo que significa que los programas escritos en Java pueden ejecutarse en cualquier dispositivo que tenga una máquina virtual Java (JVM) instalada.

  • Java SE (Standard Edition)
  • Java EE (Enterprise Edition)
  • Java ME (Micro Edition)
Versión Fecha de Lanzamiento
Java SE 8 18 de marzo de 2014
Java SE 11 25 de septiembre de 2018

Sección 2

Spring Tools

Spring Tools es un conjunto de herramientas y extensiones diseñadas para facilitar el desarrollo de aplicaciones basadas en el framework Spring para Java. Proporciona características como asistencia de codificación, generación de código, depuración, administración de dependencias y más.

  • Spring Tool Suite (STS)
  • Spring Initializr
  • Spring Boot DevTools
Versión Fecha de Lanzamiento
Spring Tools 4 4 de diciembre de 2018
Spring Tools 4.4.0 9 de diciembre de 2021

Sección 3

Java EE

Java EE (Java Enterprise Edition) es una plataforma de programación que permite desarrollar y ejecutar aplicaciones empresariales escalables y seguras en Java. Proporciona un conjunto de API y servicios para construir aplicaciones web, servicios web, aplicaciones distribuidas, y más.

  • Servlets
  • JSP (JavaServer Pages)
  • JPA (Java Persistence API)
Versión Fecha de Lanzamiento
Java EE 7 12 de junio de 2013
Java EE 8 21 de septiembre de 2017
